To install the Deadline standalone Python API, just copy the Deadline folder from this location to your site-packages folder.

To use the Deadline standalone Python API, the Deadline Webservice must be running. See the online documentation for more information on running the Deadline Webservice:
https://docs.thinkboxsoftware.com/

The Deadline standalone Python API reference documentation can be downloaded from the Deadline downloads page:
https://docs.thinkboxsoftware.com
